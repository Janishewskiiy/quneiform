Quneiform is an optical character recognition (OCR) system.
It based on cuneiform by Cognitive Technologies.

In addition to text recognition it also does layout analysis and text format recognition.
Quneiform supports recognition on following languages:
    Bulgarian
    Croatian
    Czech
    Danish
    Dutch
    English
    Estonian
    French
    German
    Hungarian
    Italian
    Latvian
    Lithuanian
    Polish
    Portuguese
    Romanian
    Russian
    Russian-English
    Serbian
    Slovak
    Spanish
    Swedish
    Turkish
    Ukrainian
