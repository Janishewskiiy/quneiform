КОНТЕЙНЕР ИЗОБРАЖЕНИЯ. CIMAGE.DLL

Цель создания

Цель создания контейнера изображений заключается в хранении, обеспечении функций доступа и модификации распознаваемого изображения.

Исходное изображение может модифицироваться в процессе распознавания (например, после бинаризации или снятия линий). Таким образом, может образовываться набор изображений.

Основная идея использования данного контейнера заключается в том, что весь набор изображений может быть сохранен в данном контейнере. При необходимости можно обращаться к любой версии исходного изображения.

Общие сведения.

Изображение идентифицируется своим именем - строкой длинной в 256 символов.

Контейнер может хранить изображения: черно-белое (1 бит на пиксел), DIB, и растровые поля.

I.               Загрузить изображение.

Bool32 PASCAL        WriteDIB(

PChar lpName - указатель на строку. Строка должна содержать имя изображения.

HANDLE lpDIB - указатель на DIB.

Word32 wCopy - флаг копирования для занесения в контейнер: если 0, то в контейнер заносится DIB, скопированный с переданного по HANDLE lpDIB, в противном случае заносится сам DIB, переданный по HANDLE lpDIB.

I.               Передать часть изображения.

Bool32 PASCAL        GetData(
                                                PChar lpName,
                                                PCIMAGE_InfoDataInGet  lpIn,
                                                PCIMAGE_InfoDataOutGet lpOut)

PChar lpName - указатель на строку. Строка должна содержать имя изображения.

PCIMAGE_InfoDataInGet lpIn - указатель на структуру с информацией о части изображения.

typedef struct CIMAGEInfoDataInGet

{

           Word32 dwX;

           Word32 dwY;

           Word32 dwWidth;

           Word32 dwHeight;

           Word16 wByteWidth;

           Word8  Reserved;

           Word8  MaskFlag;

}

CIMAGE_InfoDataInGet, * PCIMAGE_InfoDataInGet;

Word32 dwX, Word32 dwY - координаты левого верхнего угла

Word32 dwWidth, Word32 dwHeight - размеры запрашиваемого прямоугольника

Word16 wByteWidth - длина строки отводимой в буфере для передачи строки части изображения в байтах.

Word8  MaskFlag - флаг битовой маски. Если нужно получить не прямоугольное изображение, тогда флаг (не равен 0) будет указывать на то, что используется битовая маска, задающая, какие точки из заданного прямоугольника должны быть взяты из изображения. Если весь прямоугольник, тогда равен 0. Битовая маска находится непосредственно за структурой. Размер маски определяется x.

PCIMAGE_InfoDataOutGet lpOut - указатель на структуру с получаемой частью изображения.

typedef struct CIMAGEInfoDataOutGet

{

                      Word32   dwWidth;

           Word32   dwHeight;

                      Word16   wByteWidth;

           Word16   byBit;

                      PWord8   lpData;

}

CIMAGE_InfoDataOutGet, *PCIMAGE_InfoDataOutGet, **PPCIMAGE_InfoDataOutGet;

Word16   byBit - число бит на точку

Word32   dwWidth, Word32   dwHeight - размеры передаваемой части изображения.

Word16 wByteWidth - длина строки отводимой в буфере для передачи строки части изображения в байтах.

PWord8   lpData - указатель на буфер, куда будет копироваться часть изображения.

Возвращаемое значение - TRUE, в случае успеха.

I.               Заменить часть изображения.

Bool32 PASCAL        ReplaceData(

                                                PChar lpName,

                                                PCIMAGE_InfoDataInReplace lpIn)

PChar lpName - указатель на строку. Строка должна содержать имя изображения.

PCIMAGE_InfoDataInReplace lpIn - указатель на структуру с заменяемой частью изображения.

typedef struct CIMAGEInfoDataInReplace

{

                      Word16 byBit;

           Word32 dwX;

           Word32 dwY;

           Word32 dwWidth;

           Word32 dwHeight;

           Word16 wByteWidth;

           PWord8 lpData;

           Word8 Reserved;

           Word16 wReserved;

           Word8 MaskFlag;

}

CIMAGE_InfoDataInReplace, * PCIMAGE_InfoDataInReplace;

Word16 byBit - число бит на точку

Word32 dwX, Word32 dwY - координаты левого верхнего угла

Word32 dwWidth, Word32 dwHeight - размеры прямоугольника

Word16 wByteWidth - длина строки отводимой в буфере для передачи строки части изображения в байтах.

PWord8 lpData - указатель на буфер, в котором находиться заменяемая часть изображения.

Word8 MaskFlag - указатель на битовую маску, указывающую на то, какие точки из заданного прямоугольника должны быть взяты из изображения. Если весь прямоугольник, тогда  равен 0. Если нет (тогда не равен 0), то маска находится непосредственно за структурой.

Возвращаемое значение - TRUE, в случае успеха.

XVI.      Добавить область закрытую на чтение

Bool32                       AddReadCloseRects(

                                                                         PChar lpName,

                                                                         Word32 wNumberRects,

                                                                         PCIMAGE_Rect lpRects    

                                                                        )

Данная функция добавляет к изображению область, закрытую для доступа (далее маску). При получении картинки из контейнера, вместо изображения, находящегося под маской, будет выдаваться фон (подразумевается самый светлый пиксел). Маска задается прямоугольниками.

PChar lpName - Имя картинки.

Word32 wNumberRects - Колличество порямоугольников, описывающих маску.

PCIMAGE_Rect lpRects - указатель на массив структур прямоугольников. Размер массива определяется параметром wNumberRects. Структура имеет следующий вид:

typedef struct tagCIMAGERECT

{

           Word32 dwX;

           Word32 dwY;

           Word32 dwWidth;

           Word32 dwHeight;

} CIMAGE_Rect, *PCIMAGRect, **PPCIMAGE_Rect;

NOTE! Маска накладывается на изображение не во всех случаях. Это относиться к получению картинки из контейнера без копирования (см. описание ReadDIB).

Возвращаемое значение - TRUE, в случае успеха.

нужно

RemoveReadCloseRects(Name,2,Rect)

,где Rect[0]={n1,m1,nn1,mm1} Rect[1]={n2,m2,nn2,mm2}

1) добавить маску:

AddReadCloseRects(Name,2,Rect)

,где Rect[0]={k1,l1,kk1,ll1} Rect[1]={k2,l2,kk2,ll2}

XVII.    Удалить область закрытую на чтение

Bool32                       EnableMask(

                                                                         PChar lpName,

                                                                         PChar lpType,

                                                                         Bool32 bEnable    

                                                                        )

Данная функция снимает с изображения область, закрытую для доступа (далее маску). При получении картинки из контейнера, вместо изображения, находящегося под маской, будет выдаваться фон (подразумевается самый светлый пиксел). Маска задается прямоугольниками.

PChar lpName - Имя картинки.

PChar lpType – Тип маски. Принимаемые значения – “w” – маска по записи или “r” – маска по чтению.

Bool32 bEnable – переменная, блокирующая маску. TRUE – маска применяется, FALSE – игнорируется.

Возвращаемое значение - TRUE, в случае успеха.
